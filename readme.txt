Encrypt all your messages send with Contact Form 7 using a pure PHP port of GNU Privacy Guard (GPG).

== Description ==

Contact Form 7 GPG is a plugin for the well known Contact Form 7 plugin. It lets you encrypt all messages send with any Contact Form 7 form on your website using the [GPG port from Jason Hinkle](https://github.com/jasonhinkle/php-gpg). This plugin is written by [Lukas Pugstaller](http://pugstaller.com/).

<strong>Requirements: Contact Form 7 (version 3.9 or higher)</strong>

== Installation ==

1. Upload the `contact-form-7-gpg` folder to the `/wp-content/plugins/` directory
1. Activate the Contact Form 7 GPG plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to `Settings` > `Contact Form 7 GPG` in your admin menu

== Frequently Asked Questions ==

= Why does this plugin require Contact Form 7 version 3.9 or higher? =

Because the developers of Contact Form 7 changed the internal structure of their plugin and I don't want to support an old version of it.
